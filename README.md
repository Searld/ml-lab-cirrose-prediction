## Описание проекта
Данный проект реализует классификатор для предсказания исхода болезни (Status) у пациентов с циррозом печени. 

### Технологии:
- **Модель**: CatBoostClassifier
- **Оптимизация**: Optuna
- **Эксперименты**: ClearML
- **Управление зависимостями**: Poetry

## Как использовать
Проект упакован в Python-пакет.

### Установка:
```bash
pip install dist/cirrhosis_prediction-0.1.0-py3-none-any.whl
# или через poetry
poetry install
```
